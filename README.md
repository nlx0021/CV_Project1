The first project of cv.

## train——参数查找
若想进行训练的参数查找，请直接运行train.py文件。

```python train.py```
  
运行此文件后将会在以下参数中进行参数查找：1. 学习率，包括0.002、0.001和0.0005； 2. 正则化系数，包括0.01和0.001； 3. 中间层神经元个数，包括50、100和300个。
  
得到的最佳模型会自动保存在model文件夹下，名字为"My_best_model_net.pkl"
